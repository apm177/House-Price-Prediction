# House-Price-Prediction

Created a Linear Regression Model with Gradient Descent for the [House Prices - Advanced Regression Techniques Competition](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/overview) on Kaggle. The model was trained using the "train.csv" file, and the optimal weights were stored in the "weights.npy" file, the model then make predictions for the "test.csv", where it achieved a Root Mean Squared Error of 0.12875.
